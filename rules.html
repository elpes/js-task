Game rules

The objective of the game is to remove colored blocks from a game field. You have to get as much rate points as possible. 0, 1 or 2 new blocks appear after each removing. Each type of block has an unique effect:
- gives 1 rate point;
- gives 3 rate points but decreases a time by 1 second;
- gives one second but decreases a rate by 1 point;
Click 'Start' button to create blocks and start the game. Click 'New Game' to reset a timer, rate and game field. After name submition it happens automatically. After page loading it happens too. Results are stored in local storage.
